# Quark 官网 & 文档公共头部组件


本仓库更新会触发 quark-doc-home 仓库的更新，进而触发对应官网去更新。

实现所有官网的公共头部更新。


https://www.npmjs.com/package/quark-doc-header

## 如何使用

```js
npm install quark-doc-header
```


### 使用

无论是`Vue`，`React`，`Angular`还是`Jq`项目，该组件都可以被使用。

```js
// 引入
import 'quark-doc-header'

// 使用
<quark-doc-header />
```
